# Natural Company テーマ

Natural Company テーマは、夜空エトが作成しているWordPressテーマです。

主に、クライアントワーク用として作成したものとなりますので、汎用性があるテーマではありません。

使いやすい用にカスタマイズしていただくことを推奨としています。ご利用される方は、ご理解の上ご利用ください。
